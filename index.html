Innovation

Doesn't it take too long to go to calculator app and type each number?

The faster, the better. By using this app, the users doesn't need to type what was already written somewhere and can do calculations straight away from the camera. At the moment, the app is still in progress so there is a limit to what operations the users can do. Currently the Available operators are [+(addition), -(substraction), *(multiplication), /(division)]. The app will be more updated in the future with more various calculations.
